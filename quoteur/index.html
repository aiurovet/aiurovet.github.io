Version 0.1.0 

This web application allows you to create quotes as visually attractive images and save those in JPEG, PNG or SVG format.
The application neither collects private information, nor uses cookies, and doesn't even make external calls. It just uses the browser's local storage to save the preferences. Please note: this means that clearing the browser cache will discard the application's preferences too. Unfortunately, this will happen even if is installed as an app.
The application is available on any device: tablet (ideal), phone, laptop, or desktop. The recommended orientation for the handheld devices is landscape (horizontal). The only requirements are a web browser and an internet connection to start using or to get updates.

How to Install

You can use Quoteur in a browser as is without any installation. However, it can be installed on most devices via browser. This will give the native app experience and the ability to run offline when the internet connection is intermittent or unavailable. Please note that not all browsers are capable of web app installation. On handheld devices, ideally, do that with Google Chrome or Microsoft Edge on non-Apple devices, and with Safari on Apple devices. You should find and tap Add to Home screen menu item (or similar), then follow the prompts. On laptops and desktops, an icon with a monitor and an arrow might show up at the end of the address bar. When it is hovered with the mouse cursor, a tooltip will pop: Install Quoteur. Tap/click that icon and follow the prompts.

How to Use

On every start, you'll be prompted to set the phrase (the text of your quote). If you need a signature too, you can create another profile (name). That name will become the signature once you select it. You can also modify the font, size and major effects (bold, italic, underline) for both the phrase and the signature. The last but not the least, you can choose your preferred image file format. Changing this option affects all profiles.
Once the quote is ready, tap/click Download at the top of the screen and choose location/file name. The default filename is in the form of "quote_yyyy-MM-dd_HH-mm-ss.[jpg|png|svg]" that ensures uniqueness and protects the existing quotes from being overwritten by accident.

Edit

You can set or change the phrase by tapping/clicking Edit at the bottom of the screen, then typing in the new text and pressing Enter (on a keyboard of your laptop/workstation) or closing the window. If you need to insert line breaks using the keyboard of your laptop/workstation, press >Ctrl+Enter.

Profiles

You can add, rename or delete profile(s) by tapping/clicking the name of the current profile at the top right corner of the screen, then using the icons on top of the list of existing profiles in the popup dialog. The icons are (in the order of appearance): add, rename, delete, move up, move down. By deleting all profiles one by one, you delete all your preferences. No name (default) profile with the default preferences will be automatically reinstated afterwards.

Edit Profile

You can set the graphics file format for all profiles at once: JPG, PNG (default), SVG. You can set the following details for every profile separately:
  • Background colour for the whole quote.
  • Phrase font face, colour and major effects (bold, italic, underline).
  • Footer font face, colour and major effects (bold, italic, underline).
  • Footer font size ratio to the phrase font size.

Exchange

You can export and import all profiles in JSON format (copied to/from clipboard or a file). When importing, you can choose whether to append or replace the existing profiles.
