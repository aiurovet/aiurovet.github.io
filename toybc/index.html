ToyBC

Copyright © 2024 Alexander Iurovetski
This web application allows you to teach kids first words, names, birthdays and simple Math.
It is available on any device: tablet (ideal), phone, laptop or desktop. The only requirements are an internet connection and a web browser.

How to Use

On the main screen, tap or click the following arrows to navigate through the words:
  • «move to the first word
  • ‹move to the previous word.
  • ≡open the Preferences popup dialog
  • ›move to the next word
  • »move to the last word
Depending on your preferences (under ≡), tapping or clicking the current word, might submit a full or image search in another browser tab. By default, this feature is switched off, as the search results are not always child-friendly.

Preferences

You can replace the preset categories and lists of words with the ones in a language of your choice simply by updating the text in the respective text boxes: the left column for the categories, and the right one for the word lists. Once finished, close the dialogue by clicking the red Close button.
The changes to preferences you make are automatically saved to a browser's local storage, so the next time you visit the page, all these changes will not be lost. However, you can't access that storage for the sake of sharing, and if you clear your browser data, all these changes will disappear. It is highly recommended to use Export, and even better, in combination with Always checked. This will place all preferences to the clipboard as text that can be pasted to a local file or, better, to a cloud document. The latter allows you to share the preferences across devices: on another device, open that cloud document, select the text you previously pasted, switch to the ToyBC page and click Import. If, by mistake, the text you import from the clipboard is in a different format, your preferences will stay unchanged, and an error box will pop up reporting the wrong data.

Super-User Features

It is better to hide some preferences away from kids. Of course, if they can read this manual, they are capable enough to follow the instructions. But in that case, they can do things anyway. Still, for the younger kids, this approach serves as a good shield.
At the top of the Preferences screen left to the Help and Close icons, a text box is located. By typing the following commands there and closing the Preferences screen, you can do the following (the commands are case-insensitive, letters in square brackets can be omitted, in English only):
  • Reset – revert all preferences, categories and word lists to the default ones.
  • Search A[ll] – when a word is tapped or clicked, the full search on that word will be performed in the new browser tab.
  • Search I[mage] – when a word is tapped or clicked, the image search on that word will be performed in the new browser tab.
  • Search [None] – turn off the search functionality.
