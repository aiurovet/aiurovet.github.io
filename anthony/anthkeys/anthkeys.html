Language

Not all translations are complete — missing text falls back to English. To add or improve a translation, see below.

+ Contribute a translation

Open anthkeys.html, find the i18n object and langMeta right below it. Copy an existing language block, change its code (e.g. pt for Portuguese), translate all values, then add one line in langMeta. The button appears automatically.

Tip: Search for data-i18n in the HTML to find every element that shows translatable text. The attribute value (e.g. cat.editing) is the key you need to add to your translation block. Any key you don't translate will fall back to English automatically.
