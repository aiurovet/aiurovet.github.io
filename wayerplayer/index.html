This bare-minimum web application allows you to play your local audio files (MP3, AAC, OGG). That's it. Choose a folder and enjoy the sequential playback of all recognisable audio files there one after another!
The application neither collects private information, nor uses cookies, and doesn't even make external calls. It just uses the browser's local storage to save the preferences. Please note: this means that clearing the browser cache will discard the application's preferences too. Unfortunately, this will happen even if is installed as an app.
The application is available on any device: tablet (ideal), phone, laptop, or desktop. The recommended orientation for the handheld devices is landscape (horizontal). The only requirements are a web browser and an internet connection to start using or to get updates.

How to Install

You can use WayerPlayer in a browser as is without any installation. However, it can be installed on most devices via browser. This will give the native app experience and the ability to run offline when the internet connection is intermittent or unavailable. Please note that not all browsers are capable of web app installation. On handheld devices, ideally, do that with Google Chrome or Microsoft Edge on non-Apple devices, and with Safari on Apple devices. You should find and tap Add to Home screen menu item (or similar), then follow the prompts. On laptops and desktops, an icon with a monitor and an arrow might show up at the end of the address bar. When it is hovered with the mouse cursor, a tooltip will pop: Install WayerPlayer. Click that icon and follow the prompts.

How to Use

On the first start, you'll be asked to change the default player name from Anonymous to something more suitable. You can do that by tapping/clicking the modify icon (the second from the left above the list of players), then typing in any other text of your choice.
